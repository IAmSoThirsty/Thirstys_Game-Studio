# ThirstysGame - Android Application

A community-driven gaming experience built with Jetpack Compose, featuring community insights, feature proposals, and a F2P-friendly cosmetic storefront.

## Features

- **Community Insights**: View aggregated feedback from Reddit, Discord, and Steam
- **Feature Proposals**: Browse community-requested features with F2P compliance status
- **Cosmetic Storefront**: Shop for purely visual items - no pay-to-win!
- **F2P Policy**: Transparent monetization philosophy

## Tech Stack

- **UI**: Jetpack Compose with Material 3
- **Architecture**: MVVM with StateFlow
- **Navigation**: Jetpack Navigation Compose
- **Networking**: Retrofit + OkHttp + Gson
- **Testing**: JUnit, MockK, Coroutines Test

## Building the App

### Prerequisites

- Android Studio Hedgehog (2023.1.1) or later
- JDK 17
- Android SDK 34

### Debug Build

```bash
cd android/ThirstysGame
./gradlew assembleDebug
```

The debug APK will be at `app/build/outputs/apk/debug/app-debug.apk`

### Release Build

```bash
./gradlew bundleRelease
```

The release AAB will be at `app/build/outputs/bundle/release/app-release.aab`

## Running Tests

```bash
./gradlew test
```

## Play Store Deployment

### 1. Create a Keystore

```bash
keytool -genkey -v -keystore thirstysgame.keystore -alias thirstysgame -keyalg RSA -keysize 2048 -validity 10000
```

**Important**: Store the keystore file and credentials securely. Never commit them to version control.

### 2. Configure Signing for CI/CD

Add these secrets to your GitHub repository:

| Secret | Description |
|--------|-------------|
| `KEYSTORE_BASE64` | Base64-encoded keystore file |
| `KEYSTORE_PASSWORD` | Keystore password |
| `KEY_ALIAS` | Key alias (e.g., "thirstysgame") |
| `KEY_PASSWORD` | Key password |

To encode your keystore:
```bash
base64 -i thirstysgame.keystore -o keystore_base64.txt
```

### 3. Update app/build.gradle.kts for Signing

Add signing config:
```kotlin
android {
    signingConfigs {
        create("release") {
            storeFile = file(System.getenv("KEYSTORE_FILE") ?: "keystore.jks")
            storePassword = System.getenv("KEYSTORE_PASSWORD") ?: ""
            keyAlias = System.getenv("KEY_ALIAS") ?: ""
            keyPassword = System.getenv("KEY_PASSWORD") ?: ""
        }
    }
    
    buildTypes {
        release {
            signingConfig = signingConfigs.getByName("release")
            // ... rest of config
        }
    }
}
```

To change, update in:
1. `app/build.gradle.kts` - `applicationId`
2. `AndroidManifest.xml` - `package`
3. All Kotlin file package declarations

### 6. Versioning

Update in `app/build.gradle.kts`:
- `versionCode` - Integer, must increase for each release
- `versionName` - User-visible version string (e.g., "1.0.0")

## Data Source

The app loads data from JSON artifacts generated by the Python agent pipeline.

Default URL: `https://raw.githubusercontent.com/IAmSoThirsty/Thirstys_Game-Studio/main/output/app_data.json`

To use a different source, modify `AppDataRepository.kt`.

## F2P Philosophy

This app demonstrates our commitment to ethical free-to-play:

- ✅ **Cosmetic-only purchases** - No gameplay advantages
- ✅ **Transparent pricing** - No hidden costs or manipulation
- ✅ **Fair progression** - Same experience for all players
- ❌ **No loot boxes** - No gambling mechanics
- ❌ **No pay-to-win** - Skill, not wallet, determines success
